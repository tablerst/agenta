Key commands on Windows from the repo root:\n- `bun install` or `npm install`: install JS dependencies if needed.\n- `bun run dev` or `npm run dev`: run the Vite frontend only on port 1420.\n- `bun run build` or `npm run build`: run `vue-tsc --noEmit` then build the frontend bundle.\n- `bun run tauri dev` or `npm run tauri dev`: run the full desktop app in Tauri dev mode.\n- `bun run tauri build` or `npm run tauri build`: build desktop bundles.\n- `cargo check --manifest-path src-tauri/Cargo.toml`: validate Rust backend changes quickly.\n- `cargo fmt --manifest-path src-tauri/Cargo.toml`: format Rust code.\n- `cargo test --manifest-path src-tauri/Cargo.toml`: run Rust tests when backend tests exist.
