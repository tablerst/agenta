Frontend uses Vue 3 SFCs with `<script setup lang="ts">`, double-quoted imports/strings, and TypeScript strict mode (`strict`, `noUnusedLocals`, `noUnusedParameters`). Keep Vue components in PascalCase files like `App.vue`; use camelCase for refs/functions and kebab-case only in HTML attributes/CSS classes. Rust uses standard rustfmt formatting with snake_case functions like `greet` and small command-oriented Tauri functions in `src-tauri/src/lib.rs`.
