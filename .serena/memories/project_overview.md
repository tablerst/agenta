Windows-based Tauri desktop app with a Vue 3 + TypeScript frontend. Frontend source is under src/, static files under public/, and Rust/Tauri backend lives under src-tauri/ with Rust code in src-tauri/src/, permissions in src-tauri/capabilities/, generated schemas in src-tauri/gen/, and icons in src-tauri/icons/. The repository currently looks like a near-default Tauri + Vue scaffold with a greet command wired from Vue to Rust.
